# 🎙️ AI Audio Transcriber

An AI-powered full stack audio transcription application that converts speech from audio files into text using a Spring Boot backend and React frontend.

---

## 🚀 Features

- Upload audio files for transcription
- Speech-to-text conversion
- REST API integration
- Full stack architecture
- Responsive frontend interface
- Real-time transcription result display
- Audio processing support

---

## 🛠️ Tech Stack

### Backend
- Java
- Spring Boot
- Maven
- REST APIs
- Gson

### Frontend
- React.js
- Vite
- Axios
- JavaScript
- HTML/CSS
